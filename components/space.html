介绍

每个组件都需要一个私人空间

引入

import { Space, SpaceItem } from '@oasis-end/ui'

基础用法

<Space>
    <SpaceItem>1</SpaceItem>
    <SpaceItem>2</SpaceItem>
    <SpaceItem>3</SpaceItem>
</Space>

2
3

垂直

<Space vertical>

    <SpaceItem>Hello</SpaceItem>
    <SpaceItem>Hello</SpaceItem>
</Space>

API

Props

参数 说明 类型 默认值
size 间距大小 string | number | [string | number, string | number] 10
align 主轴对齐方式 ‘start’ | ‘end’ | ‘center’ | ‘baseline’ | ‘stretch’ -
justify 交叉轴对齐方式 ‘start’ | ‘end’ | ‘center’ | ‘space-around’ | ‘space-between’ -
vertical 是否垂直 boolean false
wrap 是否换行 boolean true
